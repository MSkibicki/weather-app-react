## General info
Weather app that fetches data from API to get current weather forecast.

# Technologies
Project is created with:
* React version: 16.6.3
* CSS Flexbox
* Fetch API

## Setup
To run this project, install it locally using npm:

```
$ cd ../weather-app-react
$ npm install
$ npm start
```
